Este repositório compila uma série de exercícios práticos desenvolvidos para o aprendizado da linguagem Java. O foco das atividades abrange desde os fundamentos da sintaxe até a aplicação de lógica de programação voltada para uma linguagem fortemente tipada e orientada a objetos.

---

### O que foi explorado:
* Sintaxe e Tipagem: Manipulação de variáveis, tipos primitivos e wrappers.
* Estruturas de Controle: Implementação de condicionais (if/else, switch) e laços de repetição (for, while).
* Orientação a Objetos: Introdução a conceitos de classes, objetos, construtores e métodos.
* Manipulação de Arrays e Listas: Armazenamento e processamento de coleções de dados.
* Entrada e Saída: Uso de classes como Scanner para interação via console.
